# Markdown转图片小程序

## 项目简介
这是一个微信小程序，可以将Markdown格式的文本转换成美观的长图或3:4比例的图片，方便用户在社交媒体上分享内容。

### 核心功能
- Markdown转换图片（长图/3:4比例图）
- Markdown转纯文本
- 富文本格式支持
- 表格渲染支持
- 全屏编辑功能
- 一键清空功能

## 技术架构

### 核心流程
1. **解析（Parse）**: 将Markdown文本解析成AST（抽象语法树）
2. **渲染（Render）**: 将AST转换成视觉元素
3. **导出（Export）**: 将渲染结果导出为图片

### 页面结构
- **主页面（Index）**
  - 文本输入区
  - 功能按钮区
  - 全屏编辑弹窗
  - 纯文本处理弹窗
- **图片预览页面（Preview）**
  - 图片展示区
  - 保存/分享功能

## 功能特点
- 支持复杂的Markdown语法
- 支持表格渲染（简化版，适应图片宽度限制）
- 支持一键复制纯文本
- 支持富文本格式
- 支持自定义水印
- iPad设备自适应布局

## 技术栈

# 文图小程序

一款简单易用的问答转换工具，可以将问题和回答内容转换为精美图片或纯文本。

## 功能特点

- **问答编辑**：分别编辑问题和回答内容
- **一键粘贴**：支持从剪贴板快速粘贴内容
- **全屏编辑**：提供全屏编辑模式，更加专注
- **图片转换**：将问答内容转换为精美图片
- **文本转换**：将问答内容转换为纯文本格式
- **水印选项**：可选择是否添加水印

## 页面说明

### 首页 (index)

主要编辑页面，包含问题和回答的编辑区域，以及各种功能按钮。

- 问题编辑区：输入或粘贴问题内容
- 回答编辑区：输入或粘贴回答内容
- 功能按钮：
  - 清空内容：清空所有已输入的内容
  - 转换成图片：将内容转换为图片并预览
  - 转换纯文本：将内容转换为纯文本并复制到剪贴板
  - 全屏编辑：进入全屏编辑模式

### 全屏编辑页 (fullscreen)

提供更大的编辑空间，通过标签页切换问题和回答的编辑。

- 标签页：切换问题和回答编辑
- 编辑区：更大的编辑空间
- 功能按钮：
  - 保存：保存编辑内容并返回首页
  - 取消：放弃编辑内容并返回首页

### 预览页 (preview)

展示转换后的图片效果，支持保存到相册。

- 预览区：显示转换后的图片效果
- 功能按钮：
  - 保存图片：将图片保存到相册
  - 返回：返回首页

## 使用方法

1. 在首页的问题编辑区输入或粘贴问题内容
2. 在回答编辑区输入或粘贴回答内容
3. 点击"转换成图片"按钮，进入预览页面
4. 在预览页面点击"保存图片"按钮，将图片保存到相册

或者：

1. 输入问题和回答内容
2. 点击"转换纯文本"按钮，将内容以纯文本格式复制到剪贴板

- 开发框架：微信小程序原生框架
- 开发语言：TypeScript
- 样式：WXSS
- 布局：Flex 布局
